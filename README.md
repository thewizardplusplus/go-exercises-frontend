# go-exercises-frontend

## Installation

Clone this repository:

```
$ git clone https://github.com/thewizardplusplus/go-exercises-frontend.git
$ cd go-exercises-frontend
```

Install the dependencies:

```
$ npm install
```

Build the project:

```
$ npm run build
```
